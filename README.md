# Password Strength Calculator

This function rates your password from 1 to 10.
Anton Demkin, 2017
antondemkin@yandex.ru

# Usage:

How to run:
```
python3 password_strength.py
```
example:
```
python3 password_strength.py 
Enter password: onion&garlicAREt4sty
10

python3 password_strength.py 
Enter password: monkey1999
5
```
